## Git usage
- Never remove files you did not explicitly edit during this coding session.  
- Ignore untracked or modified files that were changed outside of this session (e.g., by other Codex instances or processes).  
- Only delete files that you directly created, edited, or removed yourself.  
- Expect other Codex sessions to touch the repo in parallel; treat unexpected diffs as normal and coordinate with the user before acting.

## Reporting
- Final responses do not need to quote file paths or line numbers when summarising work.
